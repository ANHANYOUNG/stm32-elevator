# Dual Elevator

Firmware for a dual elevator system built on the STM32F103RB.

## Files
- `Elevator_103/` : STM32CubeIDE project and source files

## Environment
- MCU: STM32F103RBT6
- IDE: STM32CubeIDE
- Language: C with HAL

## Build
1. Open `Elevator_103/` in STM32CubeIDE.
2. Build the project.
3. Flash it to the board with ST-LINK.
